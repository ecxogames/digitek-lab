# Fishbot 3.0

Fishbot 3.0 is a DigiTek Lab plugin based on Fishbot 2.0.

It ports the Java version's core behavior:

- screen-region bobber detection using a dark-pixel threshold
- one-click-per-underwater-event automation
- humanized miss patterns
- timer actions
- Bloxburg fisherman income planner

Build it with:

```bash
python scripts/build.py
```

Import `release/fishbot-3.0.0.dgtkplgn` from DigiTek Lab's `Plugins > Import plugin` menu.
